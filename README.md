# Claude Code Skills Collection

这是我的 Claude Code Skills 集合。

## 包含的 Skills

| Skill 名称 | 版本 | 描述 |
|-----------|------|------|
| context-switch | v1.0.0 | 模型切换器（自动上下文恢复版） |
| github_ops | v1.0.0 | GitHub Operations Skill |
| skill-creator | v1.0.0 | 创建、修改和优化 Skills |
| skill-evolution-manager | v1.0.0 | Skills 演进管理器 |
| skill-manager | v1.0.0 | 管理和查看已安装的 Skills |
| url-to-skill | v1.0.0 | 将 GitHub 仓库或文档网站 URL 转换为 Skill |
| wechat-article-reader | v1.0.0 | 微信文章阅读器 |

## 使用方法

将需要的 skill 文件夹复制到 `~/.claude/skills/` 目录下即可使用。
